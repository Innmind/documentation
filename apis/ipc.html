#Inter Process Communication (IPC)

This is an abstraction on top of operating-system to help communicate over a socket between PHP processes.

The communication protocol has been inspired from the amqp one so it incorporates message acknowledgements and heartbeats.

Usage of this package can be found in the Silent cartographer or Kalmiya to communicate between the CLI and an HTTP server for the login process to Apple Music.
